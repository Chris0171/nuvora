Quiero desarrollar una aplicación móvil usando Flutter y Dart. 

Necesito que generes la estructura completa del proyecto siguiendo buenas prácticas de arquitectura escalable, separación de responsabilidades y código mantenible.

La aplicación es una app de productividad personal centrada en gestión de tareas diarias, notas, alarmas, recordatorios y notificaciones locales.

OBJETIVO GENERAL:

Crear una aplicación similar conceptualmente a Todoist, TickTick o Google Keep.

FUNCIONALIDADES PRINCIPALES:

1. Crear tareas diarias.
2. Editar tareas.
3. Eliminar tareas.
4. Marcar tareas como completadas.
5. Crear notas independientes.
6. Editar notas.
7. Eliminar notas.
8. Programar alarmas para tareas o notas.
9. Programar recordatorios con fecha y hora.
10. Enviar notificaciones locales al usuario.
11. Soportar tareas repetitivas (diarias, semanales, mensuales).
12. Agrupar tareas por categorías.
13. Mantener estadísticas de productividad.
14. Guardar preferencias del usuario.
15. Persistencia local con base de datos SQLite o Drift.
